This project includes the following files.
1. sxe140630_Assignment7.java which contains the java main and all calls are made from this class file.
2. sxe140630_Edge.java which is and edge class used while creating and edge and it carries the edge properties.
3. sxe140630_Graph.java which contains all the properties of the graph having adjacency list of edges and all other properties requires to make operations on graphs.
4. sxe140630_stackobject.java class file that is used to save the edges that going from and coming inside a node before shrinking the node.
5. sxe140630_Vertex.java  class file which is used to create a vertex.
6. sxe140630_Assignment7.pdf file which contains the report for this project.
